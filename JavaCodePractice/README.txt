#Java Fundamentals Code Practice

This repository contains a collection of beginner-friendly Java programs created for learning and practicing core java concepts.

## Topics Covered 
- User Input using 'Scanner' class
- User input using 'JOptionPane' dialogs
- finding the smallest number
- Calculating averages
- String manipulation (finding the middle character)
