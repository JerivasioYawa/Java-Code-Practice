#Java Fundamentals Code Practice

This repository contains a collection of beginner-friendly Java programs created for learning and practicing core java concepts.

## Topics Covered/ programs included
Program | Decription
- ScannerInputDemo |  User Input using 'Scanner' class
- JOptionPaneInputDemo | User input using 'JOptionPane' dialogs
- SmallestNumber | finding the smallest of three numbers
- AvergaeCalculator | Calculating averages of three numbers
- MiddleCharacter | String manipulation (finding the middle character of a string)

## Project structure 
src/main/java/javacodepractice/

Each class contains its own 'main' method and runs independently.

## How to run

1. Clone the repository:
  ```bash
  git clone https://github.com/JerivasioYawa/Java-Code-Practice.git

2.Open the project in your IDE (NetBeans, IntelliJ, Eclipse etc).
3. Navigate to JavaCodePractice package. 
4. Run specific code being practiced containing a main method.
